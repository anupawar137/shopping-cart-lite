# Shopping Cart Lite

A Java-based console application that simulates a basic shopping cart system.

## Features
- Add products to cart
- Remove products from cart
- View cart items
- Calculate total bill
- Simple checkout flow

## Tech Stack
- Java
- Object-Oriented Programming (OOP)

## Purpose
This project was built to practice Java fundamentals, OOP concepts, and real-world application logic.
